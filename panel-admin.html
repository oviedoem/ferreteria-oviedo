📋 Instrucciones para actualizar precios
  1. Usa el archivo del catálogo en GitHub o un CSV público
  2. Edita los valores en las columnas PrecioIVA o SocioIVA

  5. Pega el enlace en la sección 🔗 URL / Conexión
  6. Las apps cliente y vendedor actualizarán precios automáticamente

📋 Instrucciones para obtener URL pública
  1. Abre tu catálogo en GitHub o en el archivo CSV público
  2. Ve a Archivo → Compartir → Publicar en la web
  3. Selecciona la hoja con los precios
  4. En Formato selecciona CSV
  5. Haz clic en Publicar y copia el enlace
  6. Pégalo arriba en "URL pública" y haz clic en Guardar URLs
  7. Haz clic en Probar conexión para verificar
